# Labels recomendados

## Tipo
- `type:epic` — Epic (agrupa features)
- `type:feature` — Feature (agrupa tasks)
- `type:task` — Task ejecutable

## Prioridad
- `priority:P1` — Crítico
- `priority:P2` — Importante
- `priority:P3` — Opcional / mejora

## Estado
- `status:todo` — Pendiente
- `status:in-progress` — En curso
- `status:blocked` — Bloqueado
- `status:done` — Completado
